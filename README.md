# Plan Banner Generator

A Flask app for generating compact plan banners inspired by the referenced Figma site.

## Controls

- Banner Targets: `App`, `Web`
- Plan Family: `Basic`, `Infinity`, `Pro`
- Display Variants: `Light`, `Dark`
- Features

## What It Generates

- Compact multi-feature plan cards instead of full poster banners
- Live grouped previews for every selected platform and mode combination
- PNG exports for each selected combination

## Run

```bash
cd "Offering banner generator"
pip install -r requirements.txt
python app.py
```

Open `http://localhost:5000`.
